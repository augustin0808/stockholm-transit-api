# Stockholm Transit API

A FastAPI-based backend for managing transit routes in Stockholm.

## Features
- RESTful endpoints (`/status`, `/routes`, `/health`)
- SQLite database via SQLAlchemy
- Deployed on Render
- Custom domain: [api.augustincloud.org](https://api.augustincloud.org)

## Tech Stack
- Python 3.11
- FastAPI + Uvicorn
- SQLAlchemy + SQLite
- GitHub + Render + Namecheap

## Deployment
This project is deployed on [Render](https://render.com) using continuous integration from GitHub.
The API is accessible at [api.augustincloud.org](https://api.augustincloud.org).
